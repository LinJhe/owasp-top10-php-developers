# OWASP Top 10 for PHP Developers

本專案整理針對 PHP 開發人員的 OWASP Top 10 安全開發簡報，內容包含常見 Web 弱點、PHP 開發風險與安全開發實務。

## 內容重點

- OWASP Top 10 風險說明
- PHP 開發常見安全問題
- 弱點成因與攻擊情境
- Secure Coding 修補建議
- 開發人員可採取的防護措施
